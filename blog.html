Discuss the scope of var, let, and const

Global scope: Variables declared outside of any function have global scope. They can be accessed from anywhere in the code.
Function scope: Variables declared inside a function have function scope. They can only be accessed from within the function that they are declared in.
Block scope: Variables declared in a block (such as a if statement or a for loop) have block scope. They can only be accessed from within the block that they are declared in. The keywords var, let, and const are used to declare variables in JavaScript. The scope of a variable declared with each keyword is different.
var: Variables declared with the var keyword have function scope. This is the default scope in JavaScript.
let: Variables declared with the let keyword have block scope.
const: Variables declared with the const keyword also have block scope, but they cannot be reassigned once they are initialized.

What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing APIs (Application Programming Interfaces) that allows for interaction between systems. REST APIs are based on the HTTP protocol and use standard HTTP verbs (GET, POST, PUT, DELETE) to perform operations on resources.
